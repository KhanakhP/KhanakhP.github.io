Completed a comprehensive data science simulation focused on customer analytics and predictive modeling for one of the world's leading airlines. Applied end-to-end data science methodologies from web scraping and sentiment analysis to predictive modeling.

#### Key Responsibilities & Achievements:

**Customer Sentiment Analysis**
- Extracted customer reviews through web scraping techniques using Python
- Performed text preprocessing including tokenization, stopword removal, and normalization
- Applied VADER (Valence Aware Dictionary and sEntiment Reasoner) sentiment analysis to classify reviews as positive, negative, or neutral
- Generated actionable insights on customer satisfaction patterns and pain points

**Predictive Modeling for Customer Behavior**
- Developed a Random Forest classifier to predict flight booking likelihood based on customer browsing and purchase history
- Conducted comprehensive data cleaning to handle missing values, outliers, and inconsistencies
- Performed feature engineering and normalization to improve model performance
- Achieved reliable prediction accuracy for customer booking patterns

**Data Analysis & Insights**
- Analyzed customer behavior trends across multiple dimensions
- Identified key factors influencing booking decisions
- Presented findings through data visualizations and clear reporting

#### Technical Environment:
Python, Scikit-learn, Pandas, NumPy, NLTK, Beautiful Soup, Matplotlib, Seaborn

#### Skills Developed:
Web Scraping, Natural Language Processing, Sentiment Analysis, Predictive Modeling, Feature Engineering, Data Cleaning
